🎮 How to Play Simon

  • Press any key/button to start.
  • The game flashes a color (for example, Green).
  • Click that same color.
  • The game then shows the previous color and adds one more color.Example: Green → Red
  • You must repeat the entire sequence:Click Green, then Red.
  • If you click the wrong color, the game ends and you must restart.

    Round 2: 🟢 🔴 → Click 🟢 🔴
